Welcome to Hoos Around!

Hoos Around is meant for students and members of the UVA community interested finding extracurricular events on Grounds. Because it feels like so much is happening in Charlottesville at any given moment, this application serves to facilitate the process of finding events of interest. Whether you're interested in exploring music, art, food, volunteering, or just finding something to do, you have come to the right place!

Check out the View Events page to search for upcoming events, or create an event via the Create Event page for others to see!
Created events will be reviewed and approved by admins before being displayed on the map for everyone.
The Your Submitted Events page displays all events you have submitted with the respective approval status.
Please note that approved events will be visible to everyone up until one day after the event start date and time.
{% if is_admin %}
Admin users can visit the Approve Events page to manage the events pending approval or the Delete Events page to completely remove a specific event.
